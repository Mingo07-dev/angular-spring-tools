Get Catalogue Analisys by ID

This API allows user to get the score and all the values of metrics for a catalogue chosed by ID.
Try changing filters!


Get Catalogue Analisys by ID and filter by params

This API allows user to get the chosen metrics and scores for a catalogue chosed by ID.


Submit Analisys for a Catalogue already registered

This API allows user to send an RDF of a catalogue to make an analisys of it, if already exist an old analisys you must specify the ID, if not a new one will be created.
When API is call, it gives an immediate response with sintax results, if wrong it return an error, otherwhise will continue the analisys.
When analisys is finished, if an url was given, the results will be sent to the url. In any case results will be saved on the database and can later be read with the get API.
Try to use https://webhook.site to get the notification, but it can take some minutes.
